# คู่มือการใช้งานระบบจัดการก๊วนแบดมินตันแบบละเอียด (Full User & Admin Guide)

ยินดีต้อนรับสู่ระบบจัดการก๊วนแบดมินตัน! คู่มือนี้จะช่วยให้คุณใช้งานระบบได้อย่างเต็มประสิทธิภาพ ไม่ว่าคุณจะเป็นผู้เล่นหรือผู้ดูแลก๊วน

---

## 👤 สำหรับผู้เล่น (User Guide)

### 1. การเริ่มต้นใช้งาน (Getting Started)
- **การสมัครสมาชิก (Sign Up):** 
    - กรอก Email, Password และชื่อที่ต้องการให้แสดง (Display Name)
    - **สำคัญ:** ระบุชื่อ-นามสกุลจริงเพื่อความสะดวกในการยืนยันยอดชำระเงิน
    - **ระดับฝีมือ (Skill Level):** เลือกให้ตรงกับความเป็นจริงมากที่สุดเพื่อให้ระบบสุ่มแมตช์ที่สูสีและสนุกสนาน
- **การเข้าสู่ระบบ (Login):** ใช้ Email และ Password ที่สมัครไว้ หากลืมรหัสผ่านสามารถกด "Forgot Password" เพื่อรีเซ็ตผ่านอีเมลได้

### 2. กระดานคิวอัจฉริยะ (Live Board) - เมนูหลักที่ทุกคนต้องดู
หน้านี้จะแสดงข้อมูลก๊วนในวันนี้แบบ Real-time:
- **กำลังตี (Playing):** รายชื่อผู้เล่นที่กำลังลงแข่งขัน พร้อมหมายเลขคอร์ท และเวลาที่เริ่มตี
- **รอคิว (Waiting):** รายชื่อผู้เล่นที่ถูกจัดแมตช์แล้วแต่กำลังรอคอร์ทว่าง
- **จบแล้ววันนี้ (Finished Today):** ผลการแข่งขันย้อนหลังของวันนี้ พร้อมคะแนนที่เกิดขึ้น
- **สถานะคนว่าง:** ดูว่าตอนนี้ใครที่เช็คอินแล้วแต่ยังไม่มีคู่ (Waiting for match)

### 3. ระบบค่าใช้จ่ายส่วนตัว (My Expenses)
ในหน้าสีขาวขวามือ (หรือด้านล่างในมือถือ) ของกระดานคิว:
- **ยอดเงินที่ต้องชำระ:** ระบบจะคำนวณเงินให้โดยอัตโนมัติ ประกอบด้วย:
    - **ค่าสนาม:** ค่าเข้าก๊วนรายวัน (ตามที่ผู้จัดกำหนด)
    - **ค่าลูกแบด:** คำนวณจาก (ราคาต่อลูก x จำนวนลูกที่ใช้ในแมตช์ที่ได้รับมอบหมาย) 
    - *หมายเหตุ: ค่าลูกแบดจะถูกหาร 4 คนในแมตช์นั้นๆ โดยอัตโนมัติ*
- **สถานะการจ่ายเงิน (Payment Status):**
    - 🔵 **การ์ดสีน้ำเงิน:** ยังไม่จ่ายเงิน (Pending)
    - 🟢 **การ์ดสีเขียว:** จ่ายเงินเรียบร้อยแล้ว (Verified by Admin)

### 4. อันดับและสถิติ (Leaderboard)
- **ตัวกรองสถิติ:** คุณสามารถจัดอันดับตาม:
    - 🏆 **ชนะเยอะสุด:** วัดความเก่งกาจ
    - 🏸 **เล่นบ่อยสุด:** วัดความฟิต
    - 🔥 **แต้มรวมสูงสุด:** วัดพลังการทำคะแนน
    - 💸 **สายเปย์:** ยอดสะสมค่าใช้จ่ายในแต่ละเดือน
- **ช่วงเวลา:** เลือกดูได้ทั้งแบบ "ตลอดกาล" หรือแยกตาม "รายเดือน"
- **เหรียญเกียรติยศ (Badges):** รับเหรียญพิเศษ เช่น "ชนะติดต่อกัน 3 เกม" หรือ "เล่นครบ 100 เกม"

### 5. จัดการโปรไฟล์ (My Profile)
- **แก้ไขข้อมูล:** เปลี่ยนชื่อเล่น หรือปรับระดับฝีมือเมื่อคุณเก่งขึ้น
- **ประวัติบิลรายเดือน:** ดูประวัติว่าในแต่ละเดือนคุณมาตีทั้งหมดกี่วัน และรวมยอดใช้จ่ายทั้งหมดเท่าไหร่
- **การชำระเงิน:** สามารถกดดู QR Code พร้อมเพย์ของผู้จัดเพื่อชำระเงินและอัปโหลดสลิปได้ทันที

## 🔑 สำหรับผู้ดูแลก๊วน (Admin Guide)

### 1. การสร้างและจัดการก๊วน (Event Management)
- **สร้างก๊วน (New Event):** กำหนดวันที่, ยี่ห้อลูกแบด, ราคาต่อลูก และค่าสนาม
- **การเช็คอิน:** ผู้เล่นที่สมัครสมาชิกแล้วจะปรากฏให้ Admin กดเช็คอินผู้ที่มาถึงสนามแล้ว
- **สถานะก๊วน:** 
    - **เปิด (Open):** ผู้เล่นสามารถดู Live Board และสมัครเข้าร่วมได้
    - **ปิด (Closed):** เมื่อจบกิจกรรมในวันนั้น เพื่อป้องกันการแก้ไขข้อมูลย้อนหลัง

### 2. ระบบจัดแมตช์อัจฉริยะ (Matchmaker)
- **สุ่มแมตช์อัตโนมัติ (Magic Match):**
    - ระบบจะเลือกผู้เล่นที่ว่างอยู่ 4 คน โดยให้น้ำหนักกับคนที่เล่นน้อยที่สุดก่อน
    - จัดคู่โดยคำนวณจาก Skill Level เพื่อให้ทั้งสองทีมมีค่าเฉลี่ยฝีมือเท่ากันที่สุด
- **การจัดแบบแมนนวล (Manual):**
    - ลากหรือคลิกรายชื่อผู้เล่นจากแถบด้านข้าง (Sidebar) ลงทีม A หรือ B
    - ระบุเบอร์คอร์ทที่ต้องการให้ไปตี
- **การเบิกลูกแบด (+ Shuttlecock):** 
    - เมื่อผู้เล่นมาเบิก ให้กด "+ ลูกแบด" ในแมตช์นั้นและใส่เลขหมายเลขลูกแบด
    - ระบบจะบันทึกทันทีว่าลูกแบดเบอร์นี้ถูกใช้ไปแล้ว และนำไปคิดเงินผู้เล่นทั้ง 4 คนในแมตช์

### 3. การบันทึกคะแนน (Scoring)
- เมื่อแมตช์จบ ให้คลิกที่การ์ดแมตช์นั้นเพื่อใส่คะแนน
- ระบบรองรับการแข่งแบบ **2 เซต** หรือ **เซตเดียว** (ตามที่ก๊วนกำหนด)
- เมื่อบันทึกสำเร็จ ระบบจะคำนวณผลแพ้-ชนะ และอัปเดต Leaderboard ในทันที

### 4. การจัดการการเงิน (Financial Control)
- **ตรวจสอบการชำระเงิน:** ในหน้า Admin จะเห็นลิสต์รายชื่อทุกคน
- **ยืนยันการจ่ายเงิน:** เมื่อผู้เล่นชำระเงินแล้ว Admin สามารถคลิกที่ชื่อผู้เล่นเพื่อเปลี่ยนสถานะเป็น "จ่ายแล้ว ✅"
- **ภาพรวมกระแสเงินสด:** ดูยอดรวมที่ควรจะได้รับ และยอดที่เก็บจริงได้ทันทีที่ส่วนบนของหน้าจอ

### 5. สถิติเชิงลึก (Admin Stats)
- ดูผลประกอบการรายเดือน จำนวนผู้เล่นเฉลี่ย และแนวโน้มความนิยมของก๊วน เพื่อช่วยในการวางแผนจองสนามในอนาคต

---

> [!TIP]
> **เคล็ดลับ:** หากต้องการให้ตารางแข่งบน Live Board อัปเดตอยู่เสมอ Admin ควรบันทึกคะแนนทันทีที่แมตช์จบ เพื่อให้ผู้เล่นเหล่านั้นกลับมาเป็น "คนว่าง" และพร้อมสำหรับการจัดแมตช์ถัดไป!
